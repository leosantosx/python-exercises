Exercício 05

Crie uma classe para representar uma pessoa, com todos os atributos privados
nome, idade e altura.
essa classe deve ter metódos sets e gets para todos os atributos.
Crie um método para calcular a data de nascimento da pessoa.

Ela deve funcionar dessa forma:

from pessoa import Pessoa
p = Pessoa('Leo', 19, 1.69)

print(f'Nome: {p.nome} - Idade: {p.idade} - Altura: {p.altura} Data nasc: {p.data_nasc}')
print(f'Nascido no ano: {p.data_nasc}')

Imprime:
        >> Nome: Leo - Idade: 19 - Altura: 1.69
        >> Nascido no ano: 2001
